How to Use

No setup needed
Base URL http://localhost:8000
01
Pick an endpoint
Browse the endpoint list below. Each card shows the path, method, parameters, and a live example.
02
Send a GET request
Most endpoints use GET with query params. Just append ?param=value to the URL — no headers needed.
03
Get JSON back
Every response is JSON with a consistent structure: status, creator, and data.
04
Add your own plugin
Drop a .py file into plugins/{category}/ and the server auto-discovers it on restart.
